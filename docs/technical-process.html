Technical Process

Methods, Tools, and Techniques

  • UML for modeling: Used for designing both the database and application structure.
  • Typescript/JavaScript: Main programming languages for development.
    • React (via Next.js): Front-end framework for building user interfaces.
    • Sequelize: ORM (Object-Relational Mapping) for interacting with MySQL databases.
    • React Query: Library for managing server state in React applications.
  • MySQL: Relational database management system used for storing application data.
  • Git: Version control system for tracking changes in codebase and collaboration.

Software Documentation

Documentation Standards

  • The IEEE 830 standard: Used for formal requirements specification of the application.

Documentation Practices

  • Code Documentation: Inline comments and documentation to explain code functionality.
  • Technical Documentation: Detailed documentation covering system architecture, API specifications, and deployment instructions.
  • Version Control: Strict version control practices using Git to manage changes, branches, and releases.
