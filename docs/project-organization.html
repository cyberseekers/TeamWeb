Project Organization

Process Model

The project development process will follow the Unified Process (UP) methodology. It starts with gathering and analyzing requirements, followed by designing the system. Implementation of the design will be carried out, and finally, the completed product will undergo rigorous testing to identify and resolve any potential issues.

Organizational Boundaries and Interfaces

Team members will communicate via Discord meetings, ensuring regular updates and discussions. All code will be shared and managed through a GitHub repository to facilitate collaboration and version control.
