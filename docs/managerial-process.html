Managerial Process

Management Objectives and Priorities

Above all else, the main goal of the team is to keep everyone synchronized and working towards the objective of creating a user-friendly, functional search engine. We plan on meeting weekly to discuss achievements from the previous week, set goals for the current week, and address any setbacks or blockers team members are facing. By meeting frequently, we aim to maintain a high quality of work and stay on track to achieve all project goals.

Assumptions, Dependencies, and Constraints

  • Team members will organize meeting times outside of the weekly standup meeting when working together on project components.
  • Everyone has access to a laptop for remote collaboration using Discord as a primary communication tool.
  • GitHub will serve as the main repository for code storage and version control.
  • Team members are expected to use their preferred IDE for coding tasks.
  • Due to time constraints, functionality will be prioritized over aesthetic design.
