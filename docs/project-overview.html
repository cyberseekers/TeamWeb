Project Overview

Introduction

The Cyberminer, also known as the Web Search Engine, is a search engine created by software engineering students to implement Object Oriented Design concepts. The first phase of the Cyberminer will involve designing it using Object-Oriented Analysis and Design (OOAD) principles such as design patterns and UML diagrams. The next phase will take the design of the first phase and implement it using various languages and frameworks to create a Cyberminer application capable of accessing the web and searching for content using keywords provided by the user.

Objective

The objective of the Cyberminer project is to develop a functional, web-accessible search engine application. The project will involve designing the system using OOAD principles in the first phase, followed by implementation and deployment in the second phase.

Project Phases

Phase 1: Design

The first phase will focus on designing the search engine using OOAD principles, including the use of design patterns and UML diagrams. The deliverables for this phase include a detailed design document and a presentation.

Phase 2: Implementation

The second phase will involve the implementation of the design created in the first phase. This will include developing the application using appropriate programming languages and frameworks, testing the system, and preparing a final presentation and demonstration.
